PosterBoard (AKA, the lockscreen) allows you to view information like notifications, live activites and time.

Recently, various tools have come out allowing the customization of PosterBoard, specifically interactive and live wallpapers. Various examples can be found in [CAPlayground](https://caplayground.vercel.app/).
