AVKitRoutingService

AVKitRoutingService.app, a hidden app accessible via various methods, including but not limited to Stikdebug app opener.

When you open the app, there will be a message at the top reading "AVkitRoutingService Successfuly Installed!". Below, there is a message which seems to talk about an internal iOS Version where AVKitRoutingService wasn't a hidden app. It mentions the repo's README which suggests there is an internal Git Repo for this specific app. A button below shows "Kill AVKitRoutingService", an interesting detail is that it isn't Liquid Glass, which could mean this was earlier in testing or not even iOS 26 specific.
